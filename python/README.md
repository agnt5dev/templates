# Python Templates for AGNT5

Examples and templates for building AI agents with the AGNT5 Python SDK.

## Available Templates

### Quickstart
Basic example to get started with AGNT5, demonstrating core concepts like functions, entities, and workflows.

## Prerequisites

- Python 3.11 or higher
- AGNT5 SDK installed: `pip install agnt5`

## Getting Started

Each template directory contains its own README with specific instructions.
